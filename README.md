--THE PUZZLE HOUSE--
A simple puzzle game built with Python and Pygame, where players interact with letter buttons to form a correct passcode.

~ GAME OVERVIEW ~
The Puzzle House is a logic-based game where player must:
- Drag object to find hidden words
- Click on letters button
- Click ENTER to submit
- Unlock the correct solution to proceed next level
If the passcode invalid, the game will reset and allow the player to try again. There are no attempt limited for player until they find the right way.

~ FEATURES ~
- Interactive letters button
- Passcode validation system
- Automatic reset on wrong output
- Simple and clean game looping structure
- Mousebased interaction
